值类型和引用类型

值类型

直接存储的是数据本身

引用类型

存储的是数据的地址，数据单独存储的

值类型和引用类型的赋值特征

值类型

直接将变量存储的数据复制一份交给新的变量，两个变量值相同，但是存储空间完全独立，互不影响

引用类型

将变量存储的数据的地址复制一份交给新的变量，两个变量指向同一个对象，修改其中一个，另外一个也会受到影响！

值类型和引用类型参与运算的时候的规则

当值类型和引用类型做运算的时候

  1. 调用该引用类型数据的valueOf方法，获取返回值，尝试运算，如果能运算，就得出结果
  2. 如果不能，就继续调用引用类型数据的toString方法，获取返回，运算得出结果！

逻辑中断

  1. 表达式1 || 表达式2
    如果表达式1为真，则返回表达式1的值，否则，返回表达式2的值

  2. 表达式1 && 表达式2
    如果表达式1为假，则返回表达式1的值，否则，返回表达式2的值
